# VoiceMasterVolume

通話中の全員の音量に共通でかかる「マスターボリューム」を追加するユーザープラグインです。

## 使い方
- 画面左下の通話パネル（ミュート/スピーカーの並び）に追加されたボタンから調整できます。
- 設定の `通話マスターボリューム (1.0 = 100%)` でも変更可能です。
- `0.5` は 50%、`1.0` は 100%、`2.0` は 200% です。

## 注意
- Web/vesktop での通話ストリームに対して WebAudio のゲインで調整します。
- 通話の個別ユーザー音量は維持され、その上にマスターボリュームが掛かります。
